# DeFi trading app
Simple trading dApp

### Installation guide

```bash
# clone repo
$ git clone https://github.com/nenadpnc/defi-trading.git defi-trading

# change directory to your app
$ cd defi-trading

# install the dependencies with npm
$ yarn install

# build
$ yarn build

# start the server
$ yarn start
```
App is available at [http://localhost:3000](http://localhost:3000)
